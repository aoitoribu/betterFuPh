# LaTeX 入门

LaTeX 适合撰写公式密集、结构清晰的物理文档。对课程笔记来说，最值得优先掌握的是公式、交叉引用、图表和参考文献。

## 入门顺序

1. 写出行内公式和展示公式。
2. 用章节、标签和引用组织文档。
3. 插入图表并标注标题。
4. 使用 BibTeX 或 BibLaTeX 管理参考文献。

## 常用片段

```tex
\begin{equation}
  E = mc^2
\end{equation}
```
