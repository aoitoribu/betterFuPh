# 拉格朗日力学

拉格朗日力学用广义坐标描述系统，尤其适合处理约束较多或坐标选择不自然的问题。若拉格朗日量为 $L=T-V$，运动方程为

$$
\frac{d}{dt}\frac{\partial L}{\partial \dot q_i}-\frac{\partial L}{\partial q_i}=0.
$$

## 什么时候适合使用

- 约束条件较多，直接写约束力很麻烦。
- 坐标选择明显不是笛卡尔坐标。
- 系统具有可利用的对称性。
- 需要从能量结构而不是受力图出发。

::: tip
如果某个广义坐标没有显含在 $L$ 中，它对应的广义动量守恒。这个判断常常比直接求解方程更快。
:::
