# 经典力学

经典力学是大学物理训练中最适合打基础的一门课。它从受力分析和守恒律出发，逐步过渡到拉格朗日量、哈密顿量和相空间语言。掌握这些语言后，振动、刚体、连续介质和量子力学中的许多结构都会更容易理解。

## 推荐学习顺序

1. 用牛顿方程建立受力分析和微分方程习惯。
2. 用能量、动量和角动量守恒检查结果。
3. 学习虚功、约束和广义坐标。
4. 进入拉格朗日力学和哈密顿力学。
