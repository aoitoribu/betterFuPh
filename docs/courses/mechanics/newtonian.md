# 牛顿力学

牛顿力学的核心任务，是从受力分析得到运动方程。对单个质点，常见形式为

$$
\mathbf F = m\mathbf a = m\frac{d^2\mathbf r}{dt^2}.
$$

方程写出来以后，还要结合初始条件、约束条件和守恒律判断解是否合理。

## 解题步骤

1. 画出研究对象和受力图。
2. 选取坐标系，写出每个方向上的运动方程。
3. 把绳长、轨道、接触或刚体条件写成约束方程。
4. 求解后检查量纲、极限情况和守恒量。

## 常见提醒

- 不要把约束力提前假设为某个方向以外的形式。
- 非惯性系中需要明确写出惯性力。
- 守恒律不是替代受力分析，而是帮助减少方程和检查结果。
