# 电动力学

电动力学研究电磁场与带电物质的相互作用。学习时建议把矢量分析、边界条件和物理图像放在一起理解：方程给出约束，边界条件决定解的形状，物理图像帮助判断结果是否合理。

## 学习主线

1. 从静电场和静磁场建立对源、场和势的直觉。
2. 用边值问题训练格林函数、镜像法和多极展开。
3. 通过麦克斯韦方程理解变化电磁场和电磁波。
4. 在势和规范语言中重新组织方程。
