# 静磁场

静磁场由稳恒电流产生，满足

$$
\nabla \cdot \mathbf B = 0,\qquad
\nabla \times \mathbf B = \mu_0\mathbf J.
$$

第一个方程说明磁感应强度没有孤立源；第二个方程给出电流与磁场环量之间的关系。处理静磁问题时，通常先判断对称性，再选择毕奥-萨伐尔定律、安培环路定理或矢势方法。

## 典型问题

- 无限长直导线、螺线管和环形电流的磁场。
- 磁偶极矩及其远场近似。
- 矢势 $\mathbf A$ 与规范选择。
- 磁介质中的 $\mathbf B$、$\mathbf H$ 和磁化强度。
