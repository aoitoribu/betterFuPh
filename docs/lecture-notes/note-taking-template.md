# 如何整理课堂笔记

一份适合长期维护的课堂笔记，应该让读者在课后能快速找回本讲的目标、关键定义、推导线索和常见误区。不要只记录黑板上的最后公式，也要说明公式成立的条件和使用场景。

## 推荐结构

- 本讲目标：这节课解决什么问题。
- 关键定义：新符号、新概念和约定。
- 主要公式：写清楚适用条件。
- 推导步骤：保留最容易出错的中间环节。
- 常见误区：记录符号、量纲、边界条件等提醒。
- 参考资料：教材页码、讲义或补充链接。
